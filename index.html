Begrepp


Element

Kod som ska vara på hemsidan. Exempel är h3(rubrik) och p(paragraph).


HTML(generellt)

Står för HyperText Markup Language. Används för att skapa grunden till en hemsida.


ID

Används för specifiera ett element. Detta kan bara användas på ett element.


Class

Som ID, fast kan användas på flera element.


Meta (ordet)

Meta betyder data i data. T ex kan ett namn i en kod ha en egen kod.


Responsive

Responsive är att hemsidan anpassar sig utifrån skärmen. Dvs hemsidan anpassar sig om man är på mobil och dator respektive dator


Http

Står för Hypertext Transfer Protocol. Används för att överföra hemsidor till internet (WWW).


Https

Som Http, fast mer säkert med kryptering. Https hemsidor har ofta betalningsfunktioner eller liknande.


DNS

Står för Domain Name System. Förenklar adressering för IP till Internet t ex.


CDN

Står för Content Delivery Network. Används för att b la öka prestandan mellan olika datorhallar i världen.


Compressed data

Komprimerad data. Används för att spara lagring och snabba på överföringar.


Local bootstrap

Om vi har en hemsida så kommer den bara finnas lokalt på en server.


JQuery

Ett bibliotek i Javascript som är gjort att förenkla och snabba på utvecklingsprocessen av hemsidor.


Flexbox

En CSS modell som förvarar olika element i en box för att förenkla placering t ex.


Grid

En hemsidelayout som gör att man slipper använda positioning och float för att designa hemsidan.


Code snippets

Färdiga mallar för att göra det lättare att upprepa samma kod flera gånger.


WASM

Står för WebAssembly. Det är en sorts instruktioner för att samarbeta med kod och textfiler.


Chromium

En version av google med litet och snabbt gränssnitt.


svg

Står för Scalable Vector Graphics. Det är ett program för att skapa tvådimensionella bilder.


png24

Ett bildformat. PNG står för Portable Network Graphics och 24 för att formatet stödjer 24-bit färger.
